# GitHub User Finder

A web app to search any GitHub user and view their profile and repositories.

## Features
- Search any GitHub username
- Displays avatar, name, bio, followers, following, location, and public repo count
- Shows 6 most recently updated repositories with descriptions
- Click "Visit Repository" to open any repo in a new tab
- Error handling for invalid usernames
- Remembers last searched username using localStorage

## Tech Stack
- HTML, CSS, JavaScript (Vanilla)
- GitHub Public API
